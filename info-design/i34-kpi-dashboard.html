Spec — what it is, when to use it
What it is
A composite KPI dashboard: four metric tiles, a trend chart with a soft range band, and an ordered breakdown list, assembled on one twelve-column grid.
When to use it
Weekly or monthly operating reviews — the handful of numbers a team checks first, the trend behind them, and the mix that explains where the total comes from.
Data shape
{label, v, prev, series[12]}[] for the four tiles, one 12-point {v, lo, hi}[] series for the trend chart, and an unsorted {label, v}[] list that is sorted descending at render time.
Visual style
Monospace values and delta pills read at a glance; each tile's sparkline self-scales to its own range; the trend chart keeps only two gridlines and three month ticks; the breakdown list swaps a value axis for a direct number at each bar's end.

This is a composite KPI dashboard built on one twelve-column grid rather than four separate charts glued together. Four stat tiles span the top row, each showing a bold monospace value, a small pill reporting the change versus the prior period with an arrow and a color that follows its sign, and a forty-by-fourteen sparkline scaled to that one metric's own range. Below the tiles, a simplified line-and-band chart takes the left two-thirds: the accent line rides a soft translucent band showing a plausible range at every point, with only two gridlines and three month ticks to keep it quiet, and hovering it drops a crosshair, a dot, and a value chip at the nearest month. The right third holds an ordered bar list of the current breakdown, sorted largest to smallest, each row's exact share set directly at the bar's end instead of on an axis, with a soft accent tint sweeping the full row on hover. One accent color runs through every tile, the line, the band, and the bars, so the four pieces read as a single instrument.
