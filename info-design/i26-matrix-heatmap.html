Spec — what it is, when to use it
What it is
A cell grid shaded by value (sequential fill) with row and column marginal bars showing each row's and column's total.
When to use it
Relating two categorical axes at once — confusion matrices, region x product, time x channel — where both the individual cell and each axis's total matter.
Data shape
{rows[], cols[], matrix[8][6]} — row and column totals are always summed live from the matrix, never stored separately, so they can never disagree with it.
Visual style
Cell fill steps continuously from a faint to a fully saturated accent tint; only the single highest and single lowest cell carry a printed number, so the eye anchors on the extremes instead of scanning every cell; neutral ink-colored bars on the right and bottom show row and column totals.
