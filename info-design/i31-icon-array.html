Spec — what it is, when to use it
What it is
An icon-array (pictograph): a single glyph is stamped repeatedly so a rate reads as a count of figures rather than a percentage or a pie slice.
When to use it
Communicating a rate or risk to a general audience — "x in y" lands more intuitively as counted figures than as an abstract fraction.
Data shape
{k, n} for the headline rate, plus a two-group comparison pair, each expressed as k out of 10.
Visual style
One glyph defined once as an SVG <symbol> and reused everywhere; accent marks the "yes" figures, the rest recede to a flat neutral; two 1-in-10 rows do the comparison work below the main grid.

This is an icon-array pictograph: a single person glyph is defined once as an SVG symbol and stamped one hundred times into a 10-by-10 grid, with the first several figures colored in the accent hue and the remainder left a flat neutral gray, so the eye counts the accent block as a share of the whole rather than reading a percentage off an axis. The headline states the same rate in words ("about 1 in 4") instead of as a bare number, so the takeaway doesn't depend on the reader doing arithmetic. Below the main grid, the same glyph repeats in two labeled rows of ten, one per comparison group, each filled to its own count out of ten — turning "which cohort converts better" into two directly comparable little crowds instead of two bars. Hovering the main array (or either comparison row) raises a small chip restating the count as "k in n".
