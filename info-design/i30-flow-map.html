Spec — what it is, when to use it
What it is
A flow map: curved arcs run from one origin to several destinations, stroke width scaled to volume, with small arrowheads marking direction.
When to use it
Movement from a single source to multiple points — shipments, migration, network traffic — where both the destination and the relative size of each flow matter.
Data shape
COASTLINE path (same landmass + island as the symbol map) plus one origin {x,y} and {to:{x,y,label}, v}[] for 6 destinations.
Visual style
Neutral arcs for ordinary flows, one accent hero arc for the single largest destination labeled with its share of the total; a slow dash-offset drift animation reinforces the direction of travel, fully disabled under reduced motion.
