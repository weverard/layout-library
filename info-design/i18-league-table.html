Spec — what it is, when to use it
What it is
A real, sortable HTML table: rank, name, an inline value bar, and a delta pill showing change versus the prior period.
When to use it
Leaderboards, portfolio holdings, team metrics — anywhere a reader will want to re-order the list themselves, by size or by movement, rather than accept one fixed ranking.
Data shape
{name, v, prev}[] — 8 rows; delta is computed as v - prev, never stored.
Visual style
An inline accent bar scaled to each row's share of the maximum stands in for a bar-chart column inside the table; delta pills are accent for gains, red for losses; the active sort column shows a direction arrow and the rank column renumbers on every sort.

This is a real, interactive HTML table styled to match the rest of the system rather than a chart drawn to look like one. Four columns: a plain rank number, a name, a value column with a short horizontal bar built from a div scaled to that row's share of the table's largest value with the exact number set beside it, and a delta column showing a small bordered pill with an up or down arrow, colored accent for gains and a muted red for losses. Clicking the value or delta column header re-sorts the whole table by that column, toggling ascending and descending on repeated clicks, with a small arrow appearing next to the active header to show its direction. After every sort the rank column renumbers from one in the new order, and ties keep their prior relative order rather than jumping around. On narrow screens the delta column drops out entirely and the value bar shrinks, leaving rank, name, and value as the core comparison.
