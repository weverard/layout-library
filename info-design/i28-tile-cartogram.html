Spec — what it is, when to use it
What it is
A tile-grid cartogram: every region gets one equally sized tile, positioned at a fixed grid cell chosen to loosely trace the region's place on the map, rather than being scaled or shaped by true land area.
When to use it
Per-region rates or shares where physical area would otherwise dominate the reader's eye — a large rural constituency sitting next to a tiny dense one, election results, penetration or adoption rates, per-capita metrics.
Data shape
{code, label, col, row, v}[] — 24 regions, each assigned a distinct grid cell; cells left empty keep the outline reading as a coastline instead of a plain rectangle.
Visual style
A five-step sequential ramp of the accent color fills each tile by value, tiling the full range with no gaps between steps; a short code sits centered in every tile in place of a legend; the two genuine extremes each carry a leader line to a one-line label.

Hovering any tile reveals its full name and exact value in a small dark chip. On narrow screens the whole grid simply scales down as one fluid unit; it never reflows into a list, so the geographic shape stays intact at any width.
