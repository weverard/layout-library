Spec — what it is, when to use it
What it is
A funnel chart: five stacked trapezoid bands that taper continuously from the widest stage to the narrowest, each boundary's width set by the real count at that stage.
When to use it
Any process with a fixed stage order and a shrinking count at each step — signup or checkout funnels, sales pipelines, application stages.
Data shape
{label, n}[] — 5 stages in descending order; conversion percentages are computed from the counts, never stored.
Visual style
The top band is the accent color; each band below is the same hue at a lower opacity, so the ramp reads as one continuous taper rather than five unrelated colors. Stage label and count sit inside each band; conversion percentage sits in the gutter between bands, and the single steepest drop gets a red callout note.

This is a funnel chart: five trapezoid bands stacked vertically and centered on a shared axis, each one tapering smoothly from its own stage's width at the top to the next stage's width at the bottom, so the whole shape narrows continuously rather than jumping between disconnected bars. The top band is the accent color; every band below it is the same hue at progressively lower opacity, so the color itself communicates the funnel's direction without a legend. Each band carries its stage name and count directly inside it, in white or dark text depending on how saturated that band is. In the narrow gutter between each pair of bands sits the conversion percentage from one stage to the next, computed from the real counts rather than authored by hand, and the single steepest of the four drops is colored red with a short callout note explaining what happened at that step. Hovering a band lifts its opacity slightly. Under roughly 520px the tapered silhouette gives way to plain left-aligned horizontal bars, one per stage, each bar's length proportional to its count and its conversion percentage set at the bar's left edge — trading the continuous taper for a layout that stays legible at a small width.
