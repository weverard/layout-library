Spec — what it is, when to use it
What it is
A horizontal bar chart ordered strictly by value, with rank numerals in a left gutter and one bar singled out for a sentence of context.
When to use it
Straightforward "where do we stand" rankings, where the order itself is the headline and one entry — the leader, a surprise, or a laggard — deserves a callout.
Data shape
{label, v, focal?}[] — sort descending by v before rendering; flag exactly one entry focal.
Visual style
Rank numerals in a mono left gutter; direct value labels sit at each bar's end instead of a value axis; the focal bar switches to accent and carries a one-line note beneath it.

Recreate prompt
This is a horizontal bar chart with ten rows, already sorted from largest to smallest value top to bottom. A small rank numeral sits in a mono-type gutter at the far left of each row, ahead of the category label, so the order is legible even before you compare bar lengths. There is no value axis or gridlines — instead, each bar's exact value is set directly at its own trailing end in mono type, so the number and the length it belongs to are never separated by a legend or a lookup. Every bar is a neutral warm gray except one, which switches to the accent color and gains a short line of explanation set just beneath its row, connected by a small vertical accent tick — the one sentence a reader needs to understand why that entry, and not just its rank, is the point. Hovering any row washes a soft neutral tint behind the full row width, making it easy to trace a specific rank across the label, bar, and value even when the bars themselves are short.
