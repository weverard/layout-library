    Spec — what it is, when to use it
    What it is
    A single 100% stacked bar with elbow leader lines running out to a name, a percentage, and a one-line note for every segment.
    When to use it
    Single-total breakdowns where each part deserves more than a color in a legend — a budget, a headcount split, a time allocation — and there are few enough parts (four to six) that every one can carry a sentence.
    Data shape
    {label, value, note}[] — 5 segments; value is points of 100 and the set sums to exactly 100.
    Visual style
    The largest segment is computed, not hardcoded, and takes the accent color; the rest sit on a neutral ramp. Right-angle leader lines alternate above and below the bar so five labels can fan out without collisions.

    Recreate prompt
    This is a single horizontal bar, split into five segments that always sum to the whole, sitting alone in the middle of the page with generous empty space above and below it. Thin right-angle leader lines run from the center of each segment out to a label block: up for some segments, down for others, alternating so the five labels can fan out across the available width without ever colliding or needing a legend. Each label carries three things — the segment's name, its exact percentage, and one short sentence of context — so every slice of the whole gets to make its own case instead of being reduced to a number. The largest segment is identified from the data rather than assumed to be first in the list, and it alone carries the accent color and a heavier leader line; the other four sit on a plain light-to-dark neutral ramp. Hovering either a bar segment or its label block highlights both together and fades the rest, so the pairing between a shape and its sentence is never in doubt. Below roughly 560px the leader lines and floating labels disappear entirely and the same five entries reappear as a plain stacked list under the bar, each row still swappable in a hover with its segment above.
