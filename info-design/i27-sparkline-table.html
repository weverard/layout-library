Spec — what it is, when to use it
What it is
A real, sortable HTML table where each row carries its own inline sparkline alongside the current value and a delta pill.
When to use it
Watchlists and metric overviews — accounts, KPIs, portfolio positions — where the reader wants both the current number and the shape of how it got there, side by side, for several items at once.
Data shape
{name, series[12], current, prev}[] — 6 rows; the sparkline draws series, the delta pill is computed as current - prev and never stored.
Visual style
Each sparkline is a small inline SVG polyline, self-scaled to its own row's range, with a filled dot marking the current endpoint; one focal row carries a permanent accent tint; delta pills are accent for gains and a muted red for losses.

This is a real, interactive HTML table styled to match the rest of the system rather than a chart drawn to look like one. Four columns: a name, a small inline SVG sparkline scaled to that row's own range with a filled dot marking its most recent point, a current value in monospace, and a delta pill showing an up or down arrow, colored accent for gains and a muted red for losses. One row — the one the reader should be watching — carries a permanent soft accent tint on its background, distinct from the plain hover state every row gets, and that tint stays attached to the row's data wherever it lands after a sort. Clicking the Current or Δ column header re-sorts the whole table by that column, toggling ascending and descending on repeated clicks, with a small arrow appearing next to the active header to show its direction; ties keep their prior relative order rather than jumping around. The sparkline column is the point of the design, so it never disappears or shrinks away at narrow widths — if anything else has to give, the table scrolls horizontally before the trend column is sacrificed.
