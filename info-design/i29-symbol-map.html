Spec — what it is, when to use it
What it is
A proportional symbol map: a simplified coastline anchors circles at each location, with circle area — not radius — scaled to value.
When to use it
Quantities tied to specific places, where the reader needs both "where" and "how much" at once — sales by city, cases by outbreak site, inventory by depot.
Data shape
COASTLINE path (a landmass subpath + a smaller island subpath) plus {label, x, y, v}[] — 10 locations positioned on or near the coastline.
Visual style
Semi-opaque accent circles with a thin white stroke so overlapping symbols still read as distinct; only the two largest carry a direct label, the rest reveal on hover; a three-circle size key uses the identical radius scale as the map itself.

This is a proportional symbol map: a muted, hand-simplified coastline of a landmass and a smaller offshore island sits filled in a neutral tone, and accent-colored circles mark ten locations on or near it. Every circle's radius is the square root of its value times one fixed scale factor, so area — the thing the eye actually reads — encodes the quantity accurately, rather than radius doing it and silently exaggerating the largest points. A thin white stroke rings every circle so that any that overlap still read as separate shapes. Only the two largest locations carry a label directly on the map; the rest reveal their name and value in a small dark chip on hover. A three-circle size key sits in the map's bottom-left corner, each ring labeled with a round value, built from the exact same radius scale as the map symbols so it doubles as a ruler. Below roughly 560px wide, the map sits inside a letterboxed panel that keeps its true proportions, and the size key drops out of the corner to sit as its own row beneath the map.
