Spec — what it is, when to use it
What it is
A bump chart — one rounded line per entity moving up and down an inverted 1-N rank scale across periods.
When to use it
League positions, brand or product rankings — anything where relative order matters more than the underlying number.
Data shape
{label, ranks}[], ranks is number[6] with 1 = top; each period column is a permutation of 1..N.
Visual style
Entity labels sit at both ends of each line so identity survives crossings; the focal entity runs heavier in the accent color while the rest stay neutral; a softly shaded column marks the one period where the rank-1 position changed hands.
